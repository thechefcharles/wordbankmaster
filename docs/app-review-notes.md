WHAT IT IS
Players buy letters to reveal hidden phrases, play one shared Daily puzzle, build a solo "Cash Game" run, and challenge friends one-on-one or in groups.

NATIVE FUNCTIONALITY
• Push notifications via APNs (challenge results, daily reminders, social alerts) — please allow notifications when prompted.
• Full account system: sign-up, Sign in with Apple, and Google sign-in.
• Real-time multiplayer challenges with in-app chat.
• Persistent progress, streaks, leaderboards, badges, and profiles.

VIRTUAL CURRENCY (IMPORTANT)
The "Cash", "$", and balances are virtual in-game points only. They have NO real-world monetary value and cannot be purchased, sold, or withdrawn — the app has no in-app purchases. Challenge "buy-ins" are played entirely with this virtual Cash. WordBank is a skill-based word game; it is not gambling and involves no real money. This is stated in the in-app Terms of Service (Account → Terms of Service) and Privacy Policy.

ACCOUNT DELETION
Users can permanently delete their account and all associated data in-app: My Account → Delete Account (Guideline 5.1.1(v)).

2. Tap "Daily" and solve today's shared puzzle (tap letters to buy them, then "Solve").

## Tips
- Create the demo account and leave it with a little Cash/progress so the reviewer sees features immediately.
- Keep this account working through review (don't run the launch reset until after approval).
- If you get a 4.2 ("minimum functionality") question anyway, reply in Resolution Center reiterating the native push, account system, and real-time multiplayer — and that the app is a native client for a live game service, not a static website.
- If you get a gambling question, reiterate: virtual currency only, no real money, no purchases, skill-based; point to the Terms.
